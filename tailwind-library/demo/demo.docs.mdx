Import the demo component:
```bash
bit import learnbit-react.tailwind-library/demo
```

Configure the demo plugin on your workspace:

```bash
bit use demo
```

### Run the demolication

You can run your demolication on a separate port to see it outside of the Bit workspace

```bash
bit run demo
```
